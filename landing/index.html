Comprehensive Protection

Multiple layers of detection to keep your MacBook safe

Input Monitoring

Detects keyboard, mouse, and trackpad activity. Any unauthorized input triggers the alarm countdown.

Lid Close Detection

Instant alarm when the lid closes. Prevents sleep to ensure the alarm sounds even when closed.

Power Disconnect

Triggers alarm when the power cable is unplugged. Perfect protection against grab-and-run theft.

Trusted Devices

Add up to 10 trusted Bluetooth devices — iPhone, AirPods, Apple Watch, or any paired non-Apple device.

Bluetooth Proximity

Auto-disarm when any trusted device is nearby. Auto-arm when all devices leave proximity.

Touch ID + PIN

Secure authentication to disarm. Use Touch ID for convenience or your backup PIN code.

Trusted Device Auto-Disarm: When any of your trusted devices (iPhone, AirPods, Apple Watch, or non-Apple Bluetooth) is detected nearby, MacGuard automatically disarms — even while alarming. No authentication needed.

Quick Installation

Get MacGuard running in under 5 minutes

1

Download

Get the latest DMG from GitHub Releases

2

Install

Open the DMG and drag MacGuard to Applications

3

First Launch

Run in Terminal: xattr -cr /Applications/MacGuard.app

4

Grant Permission

Enable MacGuard in System Preferences → Privacy → Accessibility

System Requirements

  • macOS 13 Ventura or later
  • Accessibility permission
  • Bluetooth (for proximity)

Permissions

  • Accessibility: Input monitoring
  • Bluetooth: Device proximity
  • Admin: Lid close alarm (optional)
